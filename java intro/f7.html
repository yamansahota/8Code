Classes and It's Fundamentals :

In Java, a class is a blueprint or template for creating objects. It defines the structure and behavior of objects of that class. Classes are a fundamental concept in object-oriented programming, and they encapsulate data and methods (functions) that operate on that data. Here's an overview of how classes work in Java

1. Class Declaration :

- You declare a class using the "class" keyword followed by the class name. The class name should start with an uppercase letter, by convention.

2. Class Members :

- Inside a class, you can define various members, including fields (variables) and methods (functions). These members define the attributes and behavior of objects created from the class.

3. Object :

- To use a class, you create objects (instances) of that class. You do this by using the new keyword to instantiate the class.

4. Constructors :

- Classes can have constructors, which are special methods used to initialize objects. If you don't provide a constructor, Java provides a default constructor.

5. Access Modifiers :

- You can use access modifiers like public, private, and protected to control the visibility of class members. For example, you may want to make some members private to encapsulate the class's internal details.

6. Inhertitance :

- Java supports class inheritance, allowing you to create new classes based on existing classes. This promotes code reuse and allows you to create more specialized classes (subclasses) that inherit attributes and methods from a base class (superclass).

7. Methods and Behavior :

- Methods define the behavior of a class. You can call these methods on objects to perform actions or access data associated with the object.

8. Static Members :

- In addition to instance members, you can have static members (fields and methods) that belong to the class itself, not to individual objects. These are accessed using the class name.

9. Inner Classes :

- Java allows you to define classes inside other classes. These are called inner classes, and they are often used for organization and encapsulation.

- Classes are a fundamental building block of Java applications and are essential for creating and organizing code in an object-oriented manner.
